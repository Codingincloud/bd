How to Use These Diagrams

  1. View Online: Click "Open in Draw.io" to view and edit diagrams in your browser using the free Draw.io web app.

  3. Edit Locally: Install the free Draw.io Desktop app from diagrams.net to edit offline.
  4. Export as Image: In Draw.io, go to File → Export as → PNG (or PDF/JPEG/SVG) to create image files for your report.
  5. Print: Export as PDF for high-quality printing in your documentation.
